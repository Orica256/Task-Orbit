# Task Orbit

iPhone向けに最適化した、ローカル保存型のタスク管理Webアプリです。

## 概要

Task Orbitは、個人利用を想定したシンプルなタスク管理ツールです。  
データはブラウザのLocalStorageに保存されるため、ログイン不要ですぐ使えます。

## 主な機能

- タスクの追加・編集・削除
- 完了/未完了の切り替え
- 期限日・時刻の設定
- 優先度（高・中・低）設定
- タグ（カンマ区切り）
- 検索（タスク名・タグ）
- フィルタ（状態・優先度）
- ドラッグ&ドロップで並び替え
- セクション表示（今日 / 今週 / 期限なし）
- 進捗サマリー表示（総タスク・未完了・完了）
- ローカル保存（LocalStorage）

## 動作環境

- iPhoneのSafari（推奨）
- モダンブラウザ（Chrome / Edge / Firefox）

## 使い方

1. [index.html](index.html) をブラウザで開きます。
2. フォームにタスク内容を入力して「タスクを追加」を押します。
3. 一覧から「完了」「編集」「削除」を操作できます。
4. 上部の検索・フィルタで目的のタスクを絞り込めます。
5. タスク左側のハンドル（☰）をドラッグすると並び替えできます。

## ファイル構成

- [index.html](index.html): 画面構造（フォーム、検索、一覧）
- [styles.css](styles.css): レイアウトとデザイン
- [app.js](app.js): タスク管理ロジック、描画、保存処理

## データ保存について

- 保存先: ブラウザのLocalStorage
- キー: `task-orbit-v1`
- 同一ブラウザ・同一端末でデータが保持されます
