# Task-Orbit

モバイル用タスク管理ツールです。

## GitHub Pages 公開

- 公開URL: https://Orica256.github.io/Task-Orbit
- 公開対象: [Task Orbit](Task%20Orbit) フォルダ
- デプロイ方法: `main` へ push すると GitHub Actions で自動公開

必要ファイル:

- [.github/workflows/deploy-pages.yml](.github/workflows/deploy-pages.yml)
- [Task Orbit/.nojekyll](Task%20Orbit/.nojekyll)

GitHub側で未設定の場合は、リポジトリの `Settings > Pages > Source` を `GitHub Actions` に変更してください。
